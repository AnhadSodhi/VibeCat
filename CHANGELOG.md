# Change Log

All notable changes to the "vibecat" extension will be documented in this file.

## [1.0.2] - 2025-12-23

### Fixed
- Removed redundant release notes from README.md
### Added
- Added extension icon to package.json for marketplace display

## [1.0.1] - 2025-12-19

### Changed
- Added publisher ID to package.json
- Added GIF preview to README
- Changed version numbering scheme to be more clear

## [0.0.1] - 2025-12-16

### Added
- Initial release
- Displays GIF frames in a custom sidebar panel
- Advances to the next frame on any keyboard input (typing or cursor movement)
- Support for custom GIF files via `vibecat.customGifPath` setting
- Automatically loops back to first frame after the last frame
- Hot reload when custom GIF path is changed in settings
